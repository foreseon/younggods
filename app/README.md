## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/younggods.git
cd younggods
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
